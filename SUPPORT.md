# Getting Support

Need help with Crosscheck? Here's where to go.

## Documentation

- **Docs:** [crosscheck.cloud/docs](https://crosscheck.cloud/docs)
- **Changelog:** [crosscheck.cloud/changelog](https://crosscheck.cloud/changelog)

## Contact

| What you need | Where to go |
|---|---|
| **Product support** (bugs, install issues, account questions) | [support@crosscheck.cloud](mailto:support@crosscheck.cloud) |
| **General inquiries** (partnerships, press, business) | [info@crosscheck.cloud](mailto:info@crosscheck.cloud) |
| **Security vulnerabilities** | See [SECURITY.md](SECURITY.md) |

## Reporting bugs in Crosscheck itself

If you've found a bug in the Crosscheck product:

1. Open an issue on the relevant repository using the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md).
2. For urgent or account-blocking issues, email **support@crosscheck.cloud** directly.

## Feature requests

We welcome ideas. Open an issue using the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md) on the most relevant repository, or email **info@crosscheck.cloud** if you'd like to discuss a larger initiative.
