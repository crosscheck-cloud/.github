# .github

This repository holds organization-wide defaults for [Crosscheck](https://github.com/crosscheck-cloud).

- **`profile/README.md`** — the page shown on the [organization profile](https://github.com/crosscheck-cloud).
- **`CODE_OF_CONDUCT.md`** — applies to all Crosscheck repositories that don't define their own.
- **`CONTRIBUTING.md`** — default contribution guide for all Crosscheck repositories.
- **`SECURITY.md`** — how to report security vulnerabilities responsibly.
- **`SUPPORT.md`** — where to get help.
- **`ISSUE_TEMPLATE/`** — default issue templates inherited by repos without their own.
- **`PULL_REQUEST_TEMPLATE.md`** — default PR template.
- **`FUNDING.yml`** — sponsorship configuration.

To learn more about how this works, see GitHub's [community health files documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
